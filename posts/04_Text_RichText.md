## To Learn Flutter(04): Text, RichText

### TextDecoration

在文本附近绘制的装饰

| 属性 | 作用 |
| :------| :------ |
| underline | 下划线，在文本下面画一条线 |
| lineThrough | 删除线，在文本中间画一条线 |
| overline | 上划线，在文本上面画一条线 |

### TextDecorationStyle

控制在文本附近绘制装饰的样式

| 属性 | 作用 |
| :------| :------ |
| solid | 实线 |
| dotted | 虚线 |
| double | 两条线 |
| dotted | 点虚线 |
| wavy| 波浪线 |

### RotatedBox

顺时针旋转四分之一圈

| quarterTurns属性值 | 作用 |
| :------| :------ |
| 0 | 水平 |
| 1 | 顺时针90°翻转 |
| 2 | 顺时针180°翻转 |
| 3 | 顺时针270°翻转 |
